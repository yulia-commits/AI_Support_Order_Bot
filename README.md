# 📦 AI-ассистент техподдержки + Google Sheets

Интеллектуальный бот для автоматической проверки статуса заказа. Самый продвинутый проект в коллекции, использующий работу с API и логику на Python.

---

### 🚀 Ключевые возможности:
* **Мгновенный поиск**: Находит данные в Google Таблице по уникальному номеру заказа.
* **Python-логика**: Специальный скрипт обрабатывает данные из базы, исключая ошибки.
* **Персонализация**: Бот обращается к клиенту по имени, подтягивая его из таблицы.

---
